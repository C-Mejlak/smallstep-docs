## Name
**step api** -- authenticate to the Smallstep API

## Usage

```raw
step api <subcommand> [arguments] [global-flags] [subcommand-flags]
```

## Description

**step api** provides commands for connecting to the Smallstep API.


## Commands


| Name | Usage |
|---|---|
| **[token](token/)** | create tokens for connecting to the Smallstep API |
