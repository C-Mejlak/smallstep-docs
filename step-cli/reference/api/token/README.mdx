## Name
**step api token** -- create tokens for connecting to the Smallstep API

## Usage

```raw
step api token <subcommand> [arguments] [global-flags] [subcommand-flags]
```

## Description

**step api token** command group provides commands for creating the
tokens required to connect to the Smallstep API.


## Commands


| Name | Usage |
|---|---|
| **[create](create/)** | create a new token |
